# Subscription Gift

- **Alert key:** `kick-subscriptionGift`
- **Source:** Kick
- **Event list label:** Gifted `{{giftAmount}}` subs

## Default message template

```text
{{gifter}} gifted {{giftAmount}} {{tier}} subs to {{recipients}}
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{username}}` | `data.extraSettings.username` | The username of the sender |
| `{{avatar}}` | `data.extraSettings.avatar` | The url of the avatar of the sender for a message |
| `{{tier}}` | `data.extraSettings.tier` | Tier gift |
| `{{giftAmount}}` | `data.extraSettings.giftAmount` | Amount gifted by user |
| `{{recipients}}` | `data.extraSettings.recipients` | Users being gifted the gift subscription. Comma separated list of usernames |
| `{{recipientsRaw}}` | `data.extraSettings.recipientsRaw` | Users being gifted the gift subscription. Raw array of usernames, avatar, and userId. mainly used for custom code and custom overlays. Can be accessed with dot notation e.g. `{{recipientsRaw.0.avatar}}` to get the first users avatar |
| `{{gifter}}` | `data.extraSettings.gifter` | The username of the one who gifted |
| `{{totalGifts}}` | `data.extraSettings.totalGifts` | Gifts total count |
| `{{subMonths}}` | `data.extraSettings.subMonths` | How many months the user has been subbed for |
| `{{streakMonths}}` | `data.extraSettings.streakMonths` | How many months the user has been subbed for in a row |
| `{{message}}` | `data.extraSettings.message` | The message text sent by the user |
| `{{subPlan}}` | `data.extraSettings.subPlan` | Subs plan value (1000, 2000, 3000) |
| `{{subPlanName}}` | `data.extraSettings.subPlanName` | Subs plan value name (Tier 1, Tier 2, Tier 3) |

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `GIFT_SUB_EQUAL` - Gift subs is equal to
- `GIFT_SUB_GREATER` - Gift Subs Amount is greater than or equal to
- `EQUAL_VARIABLE`
