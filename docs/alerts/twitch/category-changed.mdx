# Category Changed

- **Alert key:** `twitch-categoryChanged`
- **Source:** Twitch
- **Event list label:** Category `{{category_name}}`

## Default message template

```text
Category changed to {{category_name}}
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{category_name}}` | `data.extraSettings.category_name` | Category name |
| `{{category_id}}` | `data.extraSettings.category_id` | Category id |
| `{{channel_title}}` | `data.extraSettings.channel_title` | Channel title |

## Example payloads

Each scenario below mirrors what Lumia emits in `Overlay.on('alert', (data) => …)`.

### Just Chatting

```json
{
  "alert": "twitch-categoryChanged",
  "dynamic": {
    "value": "Just Chatting"
  },
  "extraSettings": {
```
