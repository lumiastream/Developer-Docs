# Gifts

- **Alert key:** `youtube-gifts`
- **Source:** YouTube
- **Event list label:** Gift
- **Carries alert image:** yes (`extraSettings.contentImage`)

## Default message template

```text
{{username}} sent x{{comboCount}} {{giftName}} ({{jewelsAmount}} jewels)
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{username}}` | `data.extraSettings.username` | The username of the sender |
| `{{displayname}}` | `data.extraSettings.displayname` | The display name of the sender |
| `{{avatar}}` | `data.extraSettings.avatar` | The url of the avatar of the sender for a message |
| `{{userId}}` | `data.extraSettings.userId` | The user ID of the sender |
| `{{jewelsAmount}}` | `data.extraSettings.jewelsAmount` | No description available yet. |
| `{{totalJewels}}` | `data.extraSettings.totalJewels` | No description available yet. |
| `{{comboCount}}` | `data.extraSettings.comboCount` | No description available yet. |
| `{{giftName}}` | `data.extraSettings.giftName` | No description available yet. |
| `{{giftUrl}}` | `data.extraSettings.giftUrl` | No description available yet. |
| `{{giftImageUrl}}` | `data.extraSettings.giftImageUrl` | No description available yet. |
| `{{giftDurationSeconds}}` | `data.extraSettings.giftDurationSeconds` | No description available yet. |
| `{{giftDurationNanos}}` | `data.extraSettings.giftDurationNanos` | No description available yet. |
| `{{hasVisualEffect}}` | `data.extraSettings.hasVisualEffect` | No description available yet. |
| `{{altText}}` | `data.extraSettings.altText` | No description available yet. |
| `{{language}}` | `data.extraSettings.language` | No description available yet. |
| `{{message}}` | `data.extraSettings.message` | The remaining message after removing the command from the raw message |

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_STRING` - Gift Name is equal to
- `EQUAL_NUMBER` - Jewels is equal to
- `GREATER_NUMBER` - Jewels is greater than or equal to
- `EQUAL_USERNAME`
- `EQUAL_VARIABLE`
