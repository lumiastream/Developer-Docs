# Supersticker

- **Alert key:** `youtube-supersticker`
- **Source:** YouTube
- **Event list label:** Super Sticker
- **Carries alert image:** yes (`extraSettings.contentImage`)

## Default message template

```text
{{username}} just sent a {{stickerName}} super sticker worth {{currencySymbol}}{{amount}}
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{username}}` | `data.extraSettings.username` | The username of the sender |
| `{{displayname}}` | `data.extraSettings.displayname` | The display name of the sender |
| `{{stickerId}}` | `data.extraSettings.stickerId` | The sticker id used |
| `{{stickerName}}` | `data.extraSettings.stickerName` | The sticker name used |
| `{{amount}}` | `data.extraSettings.amount` | The amount sent |
| `{{currency}}` | `data.extraSettings.currency` | The currency used |
| `{{currencySymbol}}` | `data.extraSettings.currencySymbol` | The currency symbol used |
| `{{imageUrl}}` | `data.extraSettings.imageUrl` | No description available yet. |

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_SELECTION` - Super Sticker is equal to
- `EQUAL_CURRENCY_NUMBER` - Super Sticker Amount is equal to
- `GREATER_CURRENCY_NUMBER` - Super Sticker Amount is greater than or equal to
- `EQUAL_USERNAME`
- `EQUAL_VARIABLE`
