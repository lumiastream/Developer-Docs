# Order

- **Alert key:** `woocommerce-order`
- **Source:** WooCommerce
- **Event list label:** Order

## Default message template

```text
Someone just ordered {{item}} in the amount of {{amount}}
```

## Example payloads

Each scenario below mirrors what Lumia emits in `Overlay.on('alert', (data) => …)`.

### $100

```json
{
  "alert": "woocommerce-order",
  "dynamic": {
    "value": 100,
    "currency": "USD"
  },
  "extraSettings": {
    "username": "lumiastream",
    "item": "rgb light",
    "amount": 100,
    "currency": "USD"
  }
}
```

### $200

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_CURRENCY_NUMBER` - Donation Amount is equal to
- `GREATER_CURRENCY_NUMBER` - Donation Amount is greater than or equal to
- `EQUAL_VARIABLE`
