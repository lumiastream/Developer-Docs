# Switch Song

- **Alert key:** `youtubemusic-switchSong`
- **Source:** YouTube Music
- **Event list label:** Switch Song

## Default message template

```text
Song switched to {{name}}
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{youtubemusic_now_playing_song}}` | `data.extraSettings.youtubemusic_now_playing_song` | (Youtube Music) the current song playing |
| `{{youtubemusic_now_playing_image}}` | `data.extraSettings.youtubemusic_now_playing_image` | (Youtube Music) the url for the album art work of the current song playing |
| `{{youtubemusic_now_playing_id}}` | `data.extraSettings.youtubemusic_now_playing_id` | (Youtube Music) the id of the current song playing |
| `{{youtubemusic_now_playing_artist}}` | `data.extraSettings.youtubemusic_now_playing_artist` | (Youtube Music) the artists of the current song playing |
| `{{youtubemusic_now_playing_url}}` | `data.extraSettings.youtubemusic_now_playing_url` | (Youtube Music) the full URL of the current song playing |
| `{{youtubemusic_next_song}}` | `data.extraSettings.youtubemusic_next_song` | (Youtube Music) the next song in queue |
| `{{youtubemusic_next_image}}` | `data.extraSettings.youtubemusic_next_image` | (Youtube Music) the url for the album art work of the next song in queue |
| `{{youtubemusic_next_artist}}` | `data.extraSettings.youtubemusic_next_artist` | (Youtube Music) the artists of the next song in queue |
| `{{youtubemusic_next_id}}` | `data.extraSettings.youtubemusic_next_id` | (Youtube Music) the id of the next song in queue |
| `{{youtubemusic_next_url}}` | `data.extraSettings.youtubemusic_next_url` | (Youtube Music) the full URL of the next song in queue |
| `{{youtubemusic_queue}}` | `data.extraSettings.youtubemusic_queue` | (Youtube Music) the current queue separated by commas |
