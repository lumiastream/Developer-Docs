# Tournament Winner

- **Alert key:** `lumiastream-tournamentWinner`
- **Source:** Lumia
- **Event list label:** Tournament Winner

## Default message template

```text
Congratulations {{tournament_winner}} for winning {{tournament_title}}!
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{tournament_title}}` | `data.extraSettings.tournament_title` | Tournament title |
| `{{tournament_description}}` | `data.extraSettings.tournament_description` | Tournament description |
| `{{tournament_game}}` | `data.extraSettings.tournament_game` | Tournament game |
| `{{tournament_format}}` | `data.extraSettings.tournament_format` | Tournament format |
| `{{tournament_state}}` | `data.extraSettings.tournament_state` | Tournament state |
| `{{tournament_participants_count}}` | `data.extraSettings.tournament_participants_count` | Tournament participants count |
| `{{tournament_entry_command}}` | `data.extraSettings.tournament_entry_command` | Command used to enter the tournament |
| `{{tournament_winner}}` | `data.extraSettings.tournament_winner` | Tournament winner |
| `{{tournament_winner_avatar}}` | `data.extraSettings.tournament_winner_avatar` | Tournament winner avatar URL |
| `{{username}}` | `data.extraSettings.username` | The username of the sender |
| `{{platform}}` | `data.extraSettings.platform` | The streaming platform this message is coming from |

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_USERNAME`
- `EQUAL_VARIABLE`
