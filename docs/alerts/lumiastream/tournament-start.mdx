# Tournament Start

- **Alert key:** `lumiastream-tournamentStart`
- **Source:** Lumia
- **Event list label:** Tournament Start

## Default message template

```text
{{tournament_title}} started! Type {{tournament_entry_command}} to join the tournament
```
