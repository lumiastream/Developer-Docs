# Animation End

- **Alert key:** `vtubestudio-animationEnd`
- **Source:** Vtubestudio

## Default message template

```text
{{name}} animation ended
```

## Template variables

Use any of these in alert text, TTS, or chat templates. The same names appear under `data.extraSettings.*` when you subscribe via `Overlay.on('alert')`.

| Template variable | Overlay payload field | Description |
| --- | --- | --- |
| `{{name}}` | `data.extraSettings.name` | Name of user |
| `{{modelName}}` | `data.extraSettings.modelName` | No description available yet. |
| `{{modelId}}` | `data.extraSettings.modelId` | No description available yet. |
| `{{animationName}}` | `data.extraSettings.animationName` | No description available yet. |
| `{{animationLength}}` | `data.extraSettings.animationLength` | No description available yet. |

## Example payloads

Each scenario below mirrors what Lumia emits in `Overlay.on('alert', (data) => …)`.

### Animation End

```json
{
  "alert": "vtubestudio-animationEnd",
  "dynamic": {
    "value": "Go Bright"
  },
  "extraSettings": {
    "name": "Go Bright",
    "modelName": "Light Bulb",
    "modelId": "123",
    "animationName": "Go Bright",
    "animationLength": 40.3
  }
}
```

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_STRING` - Name is equal to
- `EQUAL_VARIABLE`
