# Recording Stopping

- **Alert key:** `meld-recordingStopping`
- **Source:** Meld

## Default message template

```text
{{name}} is stopping
```

## Example payloads

Each scenario below mirrors what Lumia emits in `Overlay.on('alert', (data) => …)`.

### Recording Stop

```json
{
  "alert": "meld-recordingStopping",
  "dynamic": {
    "value": "Go Bright"
  },
  "extraSettings": {}
}
```

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_VARIABLE`
