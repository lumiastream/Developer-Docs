# Stream Starting

- **Alert key:** `meld-streamStarting`
- **Source:** Meld

## Default message template

```text
{{name}} is starting
```

## Example payloads

Each scenario below mirrors what Lumia emits in `Overlay.on('alert', (data) => …)`.

### Stream Starting

```json
{
  "alert": "meld-streamStarting",
  "dynamic": {
    "value": "Go Bright"
  },
  "extraSettings": {}
}
```

## Variation conditions

Dimensions you can branch on when configuring alert variations:

- `RANDOM`
- `EQUAL_VARIABLE`
