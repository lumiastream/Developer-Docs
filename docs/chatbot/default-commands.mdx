# Default commands

Lumia ships with 60+ default chatbot commands seeded into every new install.

All defaults are editable in the UI. Renaming, disabling, or changing the user-level restriction works the same way as any user-created chatbot command. The seeded message is just a starting template, overwrite it freely.

## Info / utility

| Command | Behaviour |
| --- | --- |
| `!so <user>` | Shoutout with clip auto-detection. Pulls a recent clip from the target's channel. |
| `!commands` | Replies with the public commands page URL. |
| `!command` | Help for the mod-side `!command add\|edit\|delete` workflow. |
| `!uptime` | Stream uptime, with custom offline message. |
| `!followage [user]` | How long the user has been following on Twitch. |
| `!accountage [user] [platform]` | Account age for the caller or named user. Defaults to the platform the call was made on (Twitch, Kick, YouTube). |
| `!isHere` | Triggers presence detection / entrance logic. |
| `!chatstats` | Highlights a random recent chatter. |
| `!watchtime [user]` | Loyalty watch time for the caller or named user. |
| `!8ball <question>` | Random magic 8-ball answer from the classic 20. |
| `!quote` | Random quote from zenquotes.io. |
| `!when <date>` | Countdown to the supplied date. |
| `!emotes` | List the channel emotes available on the caller's platform. |
| `!emotecount <emote>` | How many times the named emote has been used since Lumia started tracking. |
| `!bot` | Show which chat sites the Lumia bot is connected to. |
| `!tip` | All-time donation total. |
| `!leaderboard` | Public loyalty leaderboard URL. |
| `!top [N]` | Top N (up to 10) loyalty point holders rendered in chat. |
| `!level [user]` | Loyalty rank for the caller or named user. |
| `!queue` | First 10 viewers in the active join queue. |

## Stream control (mod / broadcaster)

| Command | Behaviour |
| --- | --- |
| `!setgame <name>` | Updates the stream category on every connected platform that supports it (Twitch, Kick). |
| `!settitle <text>` | Updates the stream title on every connected platform (Twitch, Kick, YouTube). |
| `!alerts on\|off` / `!pause` / `!pauseAlerts` | Pause or resume Lumia alerts globally. |
| `!module <name> <enable\|disable>` | Toggle a Lumia automation by name. |
| `!timer enable\|disable <name>` | Same as `!module`, with the verb in front. |
| `!disablesfx` / `!enablesfx` | Mute / un-mute Lumia sound effects - covers TTS, HFX audio, and the general AudioManager. |

## Moderation

| Command | Behaviour |
| --- | --- |
| `!vanish` | Self-timeout for 1 second on the platform you typed from. |
| `!nuke <lookback> <action> <pattern>` | Bulk-moderate matching users across every platform they spoke on. Action is `delete`, `ban`, or a timeout duration in seconds. |
| `!nukeusername <user> [delete\|ban\|seconds]` | Bulk-moderate every recent message from a single user. |
| `!permit <user> [seconds]` | Let a user post links for N seconds (default 60) bypassing the chat filter. |
| `!filesay <url>` | Fetch a URL and post each line to chat. Capped at 50 lines, 1s/line. |

## Counters

| Command | Behaviour |
| --- | --- |
| `!editcounter <name> <value>` | Adjust a chatbot counter. Accepts absolute values or `+N`/`-N`. |

| Command | Behaviour |

## How seeding works

When Lumia launches, a one-shot seeding pass adds any missing defaults from this list. Existing commands you've customised are untouched, the pass is **additive only**. If you've deleted a default, it doesn't come back. If you renamed it, your rename stays.
