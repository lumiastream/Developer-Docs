# Event Mocks

Here are a list of mocks that can be retrieved using our Websocket events

### Twitch Chat Command

```json
{
  "origin": "twitch",
  "type": "chat/commands/twitch",
  "data": { "username": "lumiastream", "command": "blue" }
}
```
